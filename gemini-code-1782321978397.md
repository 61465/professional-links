# 📱 Local Tech Deals Aggregator (Springfield 01119)

## 🎯 رؤية المشروع
تطبيق هجين (Hybrid Aggregator) مصمم خصيصاً لجمع وعرض صفقات الأجهزة الإلكترونية (هواتف، أجهزة ألعاب، أجهزة لوحية) في النطاق الجغرافي للرمز البريدي **01119**. يركز التطبيق على جلب الأجهزة السليمة وتلك المخصصة لقطع الغيار (For Parts/Broken) وعرضها في واجهة تفاعلية تشبه "تغذية الأخبار" (Social Feed) لزيادة تفاعل المستخدمين وتسهيل اقتناص الصفقات.

---

## 🏗️ البنية التحتية والتقنيات (Tech Stack)
* **قاعدة البيانات والمصادقة:** Supabase (PostgreSQL).
* **الواجهة الخلفية ومحرك السحب (Backend/Scraper):** Python (FastAPI + BeautifulSoup/Playwright) مستضاف على Railway أو Koyeb.
* **الواجهة الأمامية (Frontend):** Flutter أو React Native (لضمان عمل الـ WebViews بكفاءة عالية على أنظمة الهاتف).
* **نمط التصميم (UI/UX):** Social Commerce Feed بألوان الأسود غير اللامع (Matte Black) والذهبي (Gold).

---

## ⚙️ الميزات الأساسية (Core Features)
1. **النهج الهجين لجلب البيانات:**
   * **مسار مباشر:** سحب البيانات من المنصات المفتوحة (Craigslist, eBay API) وتخزينها في قاعدة البيانات.
   * **مسار النوافذ المبرمجة (WebViews):** توليد روابط ديناميكية لمنصات (Facebook Marketplace, OfferUp) لتجاوز الحظر الأمني.
2. **محرك الذكاء والتصنيف (Clustering):**
   * تصنيف الأجهزة تلقائياً إلى (سليم / للقطع).
   * استخراج نوع العطل من العنوان (شاشة مكسورة، iCloud، لا يعمل).
3. **تغذية الأخبار الذكية (Smart Feed):**
   * عرض الصفقات في بطاقات تفاعلية مع تمرير لانهائي (Infinite Scrolling).
   * ترتيب الصفقات حسب الأحدث أو "أفضل صفقة" (Hot Deals).
